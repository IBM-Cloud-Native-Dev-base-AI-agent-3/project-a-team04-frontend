# project-a-team04-frontend

WIS 2026 행사 소개 페이지 형태로 구성된 React + Vite 프론트엔드 프로젝트입니다.     

## 사전 요구사항

- Node.js 18 이상 권장
- npm

## 설치

```bash
npm install
```

## 개발 서버 실행

```bash
npm run dev
```

개발 서버 정보:
- 주소: `http://localhost:4000`
- 포트: `4000`
- `strictPort: true` 설정으로, 4000 포트가 사용 중이면 다른 포트로 자동 변경되지 않고 실행이 실패합니다.

## 빌드 및 미리보기

```bash
npm run build
npm run preview
```

## 기술 스택

- React 19
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- lucide-react
- motion
